# AGENTS.md

Проект: Smetana

Это внутренний инструмент Квадрат Трубы для расчётов, КП, материалов, Telegram-бота и отчётов.

Правила работы:
1. Не менять бизнес-логику расчётов без отдельного подтверждения.
2. Не удалять существующие данные, таблицы, миграции и настройки.
3. Все изменения делать маленькими шагами.
4. Работать через отдельную ветку и pull request.
5. Перед изменениями кратко объяснять, какие файлы будут затронуты.
6. В ответах Telegram-бота не использовать слэши, markdown-код, технические символы и мусорные буквы.
7. Пользовательские сообщения должны быть простым русским текстом.
8. После изменений писать список изменённых файлов и что именно поменялось.
